West Midlands Network
Design System

Overview

The West Midlands Network Design System is a library of typography, visual styles and user interface components which are documented and updated continuously.

Use the WMN Design System to make your service/application consistent with West Midlands Network design and styles.

Our design system aims to be platform/framework agnostic as much as possible. This allows designers/developers from all practices to utilise our design system and avoid repeating work that's already been done.
